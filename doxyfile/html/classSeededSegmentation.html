Detailed Description

This class segment a given input image using the laplacian seeded segmentation proposed for Casaca et al.

See Also
Laplacian Coordinates for Seeded Image Segmentation in https://sites.google.com/site/wallacecoc/publications

Segment method receives 2 matrixes of type Mat and returns the corresponding segmentation given the input, background and foreground images.

Parameters
inputImageImage to be segmented.
backgroundImageMatrix Mat containing the background image. This matrix should contain 0 if the pixel belongs to the background and 1 otherwise.
foregroundImageMatrix Mat containing the foreground image. This matrix should contain 0 if the pixel belongs to the foreground and 1 otherwise.
neighbourhoodneighbourhood to use to calculate the laplacian matrix.
betacorresponds to a tuning constant weighting the neighborhood
sigmacorresponds to the maximum value among the all diferences. A different value can be received but it should be positive.
Returns
A binary matrix Mat containing the segmented image. This matrix will contain 0 if the pixel belongs to the foreground and 1 otherwise.
